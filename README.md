# latex_formula

Latex는 수식을 잘 쓰기 위해서 쓴다고 해도 과언이 아니다. 그래서 수식 표현을 위해 익히면 좋은 문법을 정리를 해두려고 한다.

## 구별 부호
|LaTeX|표현|
|----|---|
|`\hat{A}`|$\hat{A}$|
|`\widehat{A}`|$\widehat{A}$|
|`\tilde{A}`|$\tilde{A}$|
|`\bar{A}`|$\bar{A}$|
|`\dot{A}`|$\dot{A}$|

## 첨자
|LaTeX|표현|
|----|---|

|`A^{a,b}`|$A^{a,b}$|
|`A_{a}^{b}`|$A_{a}^{b}$|
